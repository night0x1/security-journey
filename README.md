# Security Learning Journey

## Who Am I

Just someone who decided to stop consuming and start building. I'm not from a fancy background, don't have a degree, and I'm not financially stable. But I have genuine curiosity and I'm not stopping.

This repository is my public record of learning security from the ground up — the honest version, including the struggles.

## My Approach

Every topic follows the same process:

1. Read the theory fully before touching any lab
2. Attempt the lab manually first
3. Automate with Python and requests
4. Document what I learned and why it works
5. Move on only when it genuinely clicks
